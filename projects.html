PacMen Exercise

PacMen Github Repo

    The PacMen exercise is a fun little program that starts with only two buttons on a blank page. The first button adds a pacman to the screen every time it's pressed. The second button starts the PacMen, however many have been added, moving across the screen. The PacMen move at different speeds and in different directions.

Eye Movement Exercise

Eye Github Repo

    The Eye Movement Exercise was my favorite of these projects to do, and it's fairly simple. There are two eyeballs on the page, and they follow the movement on the mouse around the screen. If the user is on a mobile device, the inner eye balls move toward where the user clicks on the screen.

Real Time Bus Tracking Exercise

Bus Tracking Github Repo

    The Real Time Bus Tracking Exercise is the most complicated of these projects. The code renders a map of the area around MIT on the page and markers denoting the locations of the active buses in that area. It works through communication with Mapbox Gl, and it automatically updates every 15 seconds.
